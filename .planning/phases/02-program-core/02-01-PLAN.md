<task type="auto">
  <name>Task 2: Implement full Anchor program in lib.rs (all 3 instructions + PDAs + errors)</name>
  <files>programs/bay-mileage-store/src/lib.rs</files>
  <action>
    Replace the entire contents of programs/bay-mileage-store/src/lib.rs with the following implementation. This replaces the placeholder `initialize` instruction from Phase 1.

    Key design decisions to implement (all from research):

    1. **StoreConfig PDA** — seeds `[b"store_config"]`, stores `authority: Pubkey`, `bay_mint: Pubkey`, `bump: u8`. Storing `bay_mint` in config addresses the open question from research (prevents mint substitution attacks via `address = store_config.bay_mint` constraint in Purchase).

    2. **StoreItem PDA** — seeds `[b"item", name.as_bytes()]`, stores `name: String` (max 32 chars via `#[max_len(32)]`), `price: u64` (raw BAY units, 10^6 per BAY), `stock: u64`, `bump: u8`. Use `#[derive(InitSpace)]` to avoid manual space calculation.

    3. **PurchaseReceipt PDA** — seeds `[b"receipt", buyer.key().as_ref(), item.key().as_ref()]`, stores `buyer: Pubkey`, `item: Pubkey`, `amount_burned: u64`, `timestamp: i64` (note: `i64` not `u64` — matches `Clock::get()?.unix_timestamp` return type), `bump: u8`.

    4. **initialize_store instruction** — sets `store_config.authority = authority.key()`, `store_config.bay_mint = bay_mint.key()`, `store_config.bump = ctx.bumps.store_config`. Takes `authority: Signer` (mut, payer) and `bay_mint: Account<'info, Mint>` (unchecked read — just recording its address).

    5. **add_item instruction** — validates `name.len() <= 32` before using name as seed, sets item fields. Requires `has_one = authority` on `store_config`. Uses `#[instruction(name: String)]` above the Accounts derive.

    6. **purchase instruction** — sequence: (a) check `buyer_token_account.amount >= item.price` via `require_gte!`, (b) check `item.stock > 0`, (c) decrement stock, (d) CPI burn via `token::burn`, (e) initialize receipt PDA fields. The `bay_mint` account uses `address = store_config.bay_mint` constraint. The `item` account uses `seeds = [b"item", item.name.as_bytes()], bump = item.bump` to re-verify it is a legitimate registered item (not a fake account).

    7. **BayError enum** — three variants: `InsufficientFunds`, `OutOfStock`, `NameTooLong`.

    8. **Bump API** — use `ctx.bumps.account_name` (field access, NOT `.get("account_name")` — that is the deprecated Anchor 0.28-era API).

    Write the complete file:

    ```rust
    use anchor_lang::prelude::*;
    use anchor_spl::{
        associated_token::AssociatedToken,
        token::{self, Burn, Mint, Token, TokenAccount},
    };

    declare_id!("3HgyGbc6oQvGMiWxMFCTfk4br1K4Zv6Jy2oR3chCcFag");

    #[program]
    pub mod bay_mileage_store {
        use super::*;

        pub fn initialize_store(ctx: Context<InitializeStore>) -> Result<()> {
            let config = &mut ctx.accounts.store_config;
            config.authority = ctx.accounts.authority.key();
            config.bay_mint = ctx.accounts.bay_mint.key();
            config.bump = ctx.bumps.store_config;
            Ok(())
        }

        pub fn add_item(
            ctx: Context<AddItem>,
            name: String,
            price: u64,
            stock: u64,
        ) -> Result<()> {
            require!(name.len() <= 32, BayError::NameTooLong);
            let item = &mut ctx.accounts.item;
            item.name = name;
            item.price = price;
            item.stock = stock;
            item.bump = ctx.bumps.item;
            Ok(())
        }

        pub fn purchase(ctx: Context<Purchase>) -> Result<()> {
            let item_price = ctx.accounts.item.price;
            let item_stock = ctx.accounts.item.stock;
            let buyer_amount = ctx.accounts.buyer_token_account.amount;

            require_gte!(buyer_amount, item_price, BayError::InsufficientFunds);
            require!(item_stock > 0, BayError::OutOfStock);

            ctx.accounts.item.stock -= 1;

            let cpi_ctx = CpiContext::new(
                ctx.accounts.token_program.to_account_info(),
                Burn {
                    mint: ctx.accounts.bay_mint.to_account_info(),
                    from: ctx.accounts.buyer_token_account.to_account_info(),
                    authority: ctx.accounts.buyer.to_account_info(),
                },
            );
            token::burn(cpi_ctx, item_price)?;

            let receipt = &mut ctx.accounts.receipt;
            receipt.buyer = ctx.accounts.buyer.key();
            receipt.item = ctx.accounts.item.key();
            receipt.amount_burned = item_price;
            receipt.timestamp = Clock::get()?.unix_timestamp;
            receipt.bump = ctx.bumps.receipt;

            Ok(())
        }
    }

    // ── Account Structs ─────────────────────────────────────────────────────────

    #[account]
    #[derive(InitSpace)]
    pub struct StoreConfig {
        pub authority: Pubkey, // 32 — operator wallet
        pub bay_mint: Pubkey,  // 32 — BAY token mint address
        pub bump: u8,          // 1
    }

    #[account]
    #[derive(InitSpace)]
    pub struct StoreItem {
        #[max_len(32)]
        pub name: String,  // 4 + 32 = 36
        pub price: u64,    // 8 — raw BAY units (1 BAY = 1_000_000 units)
        pub stock: u64,    // 8
        pub bump: u8,      // 1
    }

    #[account]
    #[derive(InitSpace)]
    pub struct PurchaseReceipt {
        pub buyer: Pubkey,       // 32
        pub item: Pubkey,        // 32
        pub amount_burned: u64,  // 8 — raw BAY units burned
        pub timestamp: i64,      // 8 — Clock::get()?.unix_timestamp (i64, not u64)
        pub bump: u8,            // 1
    }

    // ── Instruction Contexts ─────────────────────────────────────────────────────

    #[derive(Accounts)]
    pub struct InitializeStore<'info> {
        #[account(
            init,
            payer = authority,
            space = 8 + StoreConfig::INIT_SPACE,
            seeds = [b"store_config"],
            bump,
        )]
        pub store_config: Account<'info, StoreConfig>,

        pub bay_mint: Account<'info, Mint>,

        #[account(mut)]
        pub authority: Signer<'info>,

        pub system_program: Program<'info, System>,
    }

    #[derive(Accounts)]
    #[instruction(name: String)]
    pub struct AddItem<'info> {
        #[account(
            init,
            payer = authority,
            space = 8 + StoreItem::INIT_SPACE,
            seeds = [b"item", name.as_bytes()],
            bump,
        )]
        pub item: Account<'info, StoreItem>,

        #[account(
            seeds = [b"store_config"],
            bump = store_config.bump,
            has_one = authority,
        )]
        pub store_config: Account<'info, StoreConfig>,

        #[account(mut)]
        pub authority: Signer<'info>,

        pub system_program: Program<'info, System>,
    }

    #[derive(Accounts)]
    pub struct Purchase<'info> {
        #[account(mut)]
        pub buyer: Signer<'info>,

        #[account(
            mut,
            associated_token::mint = bay_mint,
            associated_token::authority = buyer,
        )]
        pub buyer_token_account: Account<'info, TokenAccount>,

        #[account(
            mut,
            address = store_config.bay_mint,
        )]
        pub bay_mint: Account<'info, Mint>,

        #[account(
            seeds = [b"store_config"],
            bump = store_config.bump,
        )]
        pub store_config: Account<'info, StoreConfig>,

        #[account(
            mut,
            seeds = [b"item", item.name.as_bytes()],
            bump = item.bump,
        )]
        pub item: Account<'info, StoreItem>,

        #[account(
            init,
            payer = buyer,
            space = 8 + PurchaseReceipt::INIT_SPACE,
            seeds = [b"receipt", buyer.key().as_ref(), item.key().as_ref()],
            bump,
        )]
        pub receipt: Account<'info, PurchaseReceipt>,

        pub token_program: Program<'info, Token>,
        pub associated_token_program: Program<'info, AssociatedToken>,
        pub system_program: Program<'info, System>,
    }

    // ── Custom Errors ────────────────────────────────────────────────────────────

    #[error_code]
    pub enum BayError {
        #[msg("BAY token balance is insufficient for this purchase")]
        InsufficientFunds,

        #[msg("Item is out of stock")]
        OutOfStock,

        #[msg("Item name must be 32 characters or fewer")]
        NameTooLong,
    }
    ```

    Notes on choices made:
    - `bay_mint` stored in `StoreConfig` so `Purchase` can verify `address = store_config.bay_mint` — prevents passing a counterfeit mint
    - `item` in `Purchase` has full seed re-derivation (`seeds = [b"item", item.name.as_bytes()], bump = item.bump`) — prevents fake StoreItem accounts
    - `stock` decremented BEFORE burn CPI — if CPI fails, the Solana runtime rolls back the entire transaction, so the decrement is safe
    - One-receipt-per-buyer-per-item limitation is intentional for Phase 2 (documented in research)
    - Price is stored as raw BAY units (multiply whole BAY by 1_000_000 before passing to add_item)
