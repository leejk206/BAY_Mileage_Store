<task type="auto">
  <name>Task 2: Create BAY SPL Token and mint to test wallet</name>
  <files>
    wallets/test-wallet.json
    .env
  </files>
  <action>
    1. Configure Solana CLI for devnet:
       ```bash
       solana config set --url devnet
       ```

    2. Verify deployer wallet exists (`~/.config/solana/id.json`). If not, generate one:
       ```bash
       solana-keygen new --outfile ~/.config/solana/id.json
       ```

    3. Fund deployer wallet with SOL:
       ```bash
       solana airdrop 2
       solana balance
       ```
       If airdrop fails (rate-limited), instruct user to visit https://faucet.solana.com/ and fund the address shown by `solana address`.

    4. Create BAY mint with 6 decimals:
       ```bash
       spl-token create-token --decimals 6
       ```
       Capture and save the MINT_ADDRESS from the output.

    5. Create deployer's token account for BAY:
       ```bash
       spl-token create-account <MINT_ADDRESS>
       ```

    6. Mint 1,000,000 BAY to the deployer's token account:
       ```bash
       spl-token mint <MINT_ADDRESS> 1000000
       ```

    7. Generate a separate test wallet:
       ```bash
       mkdir -p wallets
       solana-keygen new --outfile wallets/test-wallet.json --no-bip39-passphrase
       ```

    8. Fund test wallet with SOL (needed for token account rent):
       ```bash
       solana airdrop 1 $(solana-keygen pubkey wallets/test-wallet.json)
       ```

    9. Create test wallet's BAY token account and mint BAY to it:
       ```bash
       spl-token create-account <MINT_ADDRESS> --owner $(solana-keygen pubkey wallets/test-wallet.json)
       ```
       Then mint 100,000 BAY to the test wallet's token account:
       ```bash
       spl-token mint <MINT_ADDRESS> 100000 <TEST_WALLET_TOKEN_ACCOUNT>
       ```
       Where TEST_WALLET_TOKEN_ACCOUNT is the address output from the create-account command above.

    10. Save key addresses to `.env` file at project root:
        ```
        BAY_MINT=<MINT_ADDRESS>
        TEST_WALLET_PUBKEY=<pubkey from wallets/test-wallet.json>
        DEPLOYER_PUBKEY=<pubkey from solana address>
        ```

    11. Add `.env` to `.gitignore` if not already present. Add `wallets/` to `.gitignore` to protect the test wallet keypair.

    12. Verify test wallet BAY balance:
        ```bash
        spl-token balance <MINT_ADDRESS> --owner $(solana-keygen pubkey wallets/test-wallet.json)
        ```
        Should show 100000.
  </action>
  <verify>
    - `spl-token balance <MINT_ADDRESS>` shows 1000000 (deployer)
    - `spl-token balance <MINT_ADDRESS> --owner $(solana-keygen pubkey wallets/test-wallet.json)` shows 100000
    - `wallets/test-wallet.json` file exists
    - `.env` file contains BAY_MINT, TEST_WALLET_PUBKEY, DEPLOYER_PUBKEY
    - `.gitignore` includes `.env` and `wallets/`
  </verify>
  <done>
    BAY SPL Token (decimals=6) exists on Solana devnet. The deployer wallet holds 1,000,000 BAY. The test wallet (wallets/test-wallet.json) holds 100,000 BAY. Mint authority is retained on the deployer keypair.
  </done>
